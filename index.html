Intro

This guide lists the components of FXStreet's main website.

The project is built with LESS and compiled into one single CSS file for the main site.

Aside from the custom LESS files, we also incorporate vendor components from Bootstrap v3.3.5, LESS mixins from Less Hat and icons from Fontawesome.

The Project's structure

The components are divided with the Atomic design aproach.

In order to ensure mantainance and scalability, each group of components are divided into different LESS files to import all into the final "fxs_screen.min.css" or use it separately in order to build third partie widgets.

In order to get started, you have to know the structure of the project, the LESS files and the compiled CSS files
